Neuro-Moodle-LLM

Where course content meets retrieval intelligence

Self-hosted Moodle 5.2, PostgreSQL 16, Qdrant, and Ollama — with a Python layer that embeds course material and answers in context.

Moodle 5.2 PostgreSQL Qdrant Ollama Python Docker

A custom-built Moodle image (official tarball on pinned PHP 8.3 + Apache), not a third-party distro. The CLI neuro-moodle-llm calls Moodle web services, pushes vectors to Qdrant, and grounds answers in llama3.2 with nomic-embed-text embeddings.

Full setup, architecture diagrams, and commands live in the repository README on GitHub.

Quick start

cp .env.example .env
docker compose up -d --build
docker compose exec ollama ollama pull llama3.2
docker compose exec ollama ollama pull nomic-embed-text
